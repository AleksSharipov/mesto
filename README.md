# Проект 4: Место

### Обзор

В проекте реализован полноценный одностраничный сайт, с шапкой, подвалом, карточками мест и всем сопутствующем контентом. Так же использовалось большое кол-во технологий.

Использовалась:

* flex секта;
* grid-layout;
* БЭМ;
* БЭМ Nested
* адаптивная верстка
* реализован попап с помощью JS
